# Alex_DataAnalyst_Bootcamp_Projects

This repository contains projects and assignments completed as part of the Data Analyst Bootcamp by Alex the Analyst. The bootcamp covers data analysis using Excel, SQL, and Python.

## Repository Structure

- data/: Contains sample data files used for analysis.
  - sample_data.xlsx: An Excel file with sample data.
- reports/: Contains the final report of the project.
  - data_analysis_report.pdf: The final report in PDF format.
- presentations/: Contains the presentation files.
  - data_analysis_presentation.pptx: The final presentation in PowerPoint format.
- scripts/: Contains the Python scripts used for data analysis.
  - data_cleaning.py: Script for data cleaning.
  - data_analysis.py: Script for data analysis.
  - data_visualization.py: Script for data visualization.
- SQL_queries/: Contains SQL queries used in the project.
  - data_analysis_queries.sql: SQL file with data analysis queries.
